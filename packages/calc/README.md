# Math and arrays calculation helpers

## `round`

```javascript

round(2.98, 0); // => 3
round(4005, 4); // => 4005.0000
```

## `sum`

```javascript

sum([1, 2, 3, 4]); // => 10
sum([6, 12, 4]); // => 22
```

## `average`

```javascript

average([1, 2, 3, 4]); // => 2.5
average([6, 12, 4]); // => 7.33
```

## `group`

```javascript

gruup([1, 2, 3, 4], 2); // => [[1, 2], [3, 4]]
average([6, 12, 4], 2); // => [[6, 12], [4]]
```
